*1:  
```
( 2 * neck_top_width * thumb_to_thumb + neck_top_width * neck_bot_width ) / ( 4 * thumb_to_thumb + 4 * neck_bot_width - 2 * neck_top_width )
```  

*2:  
```
neck_bot_width / 2
```
